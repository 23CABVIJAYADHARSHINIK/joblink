# JobLink 

JobLink is a web application that helps users analyze resumes and find matching job opportunities.

##  Features
- Resume analysis
- Job matching based on skills
- Simple and clean UI
- Frontend built using React + Vite

##  Tech Stack
- React
- Vite
- HTML
- CSS
- JavaScript
